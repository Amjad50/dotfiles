# Thunderbird extensions — install manually per machine

Copying `.xpi` files doesn't work: Thunderbird re-packs installed add-ons
unsigned, so a copied `.xpi` lands `appDisabled` (unsigned) and won't load on a
release build. So on each machine, download the signed `.xpi` from the store URL
below (or search the store UI) and install via Settings → Add-ons and Themes.

`tb-sync load` prints these same URLs so you can ctrl-click each one.

After installing **tbkeys-lite**: Add-ons → tbkeys-lite → Options → paste the
contents of `~/.thunderbird/tbkeys-keymap.json` into the keybindings box → Save.
